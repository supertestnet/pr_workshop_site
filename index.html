Bitcoin Script Workshop

Learn to use Bitcoin Script

In this workshop, renowned bitcoin educator Super Testnet will teach you how to write smart contracts using the Bitcoin Script programming language. Each participant will create a simple smart contract called an Oracle, based on a fictional upcoming sporting event, where two people bet a certain amount on the outcome, and the winner gets the sum.

By writing this simple app from scratch, you'll learn the basics of programming in a bitcoin environment, how to use the blockchain, and can carry these skills forward for a prospective future as a bitcoin developer. Please bring a standard laptop to participate in the workshop -- nothing else is required!

When: Saturday, March 21 at 2:00 pm

Where: Maria Guevara Library @ Sagrado Corazón University, Estudio Room

Note: No food or drinks are permitted, but there is a weekly "Bitcoin Brunch" every Saturday (including this one) at 11 am, organized by the same people, though at a different location. You are welcome to attend that and then head over to the workshop afterwards. More details here: brunch.supertestnet.org

Cost: 15,000 sats (about $10), to be reimbursed during the workshop
